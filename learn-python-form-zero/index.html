Python语言的特点

  • 学习编程不一定要做开发工程师，它会教会你一种思考方式。编程是一种技能和工具，是你理解计算机运行的最有效的途径。——乔布斯
  • 把Python当做你的一个工具。
  • Python语言的特点：
    1. 语法简洁
    2. 学习梯度可接受
    3. 跨平台，Windows、Linux、Mac
    4. 可扩展性，扩展C、C++
    5. 开放源代码
    6. 类库丰富
