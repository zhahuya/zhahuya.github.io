权利

014-你的权利从哪里来？

  • 权利的起源、定义、特征、保护、行使和治理的结构
  • 过去主要关注一些宏观经济指标，现在是对于经济组织内部结构的分析

权利有别于能力

  • 权利是文明的产物。//动物的竞争是直接的竞争，弱肉强食。
  • 权利（财产权利）的定义：产权是一种通过社会强制而实现的，对某种经济物品的多种用途进行选择的权利。（A property right is a socially enforced right to select uses of an economic good.）
  • 权利是要由社会来保障实行的，是别人对你的认可，不是你自己认为有就有，是别人认为你有，别人愿意出来保护你形式的，那才叫权利。
  • 权利有别于能力（Might or Right）

权利是人赋的而非天赋的

  • 能力取决于你自己能够占有多少，而权利取决于社会愿意给你多少，维护你多少。
  • “人赋权利”是一种实证观点。//规范的权利和实证的权利。
  • 《你的权利从哪里来？》Right from Wrongs: A Secular Theory of the Origins of Rights, “权利来自错误”
    • 权利不来自于神。
    • 权利不来自自然
    • 权利不来于逻辑
    • 权利不来自法律
    • 权利来自于人类的经验教训
  • 权利需要社会上其他人的共同认可和执行

自由不等于免费

  • 财产权利和政治权利同等重要
  • 行使权利所要动用的经济资源并不免费
  • 经济权利是人的基本权利

产权保护物理属性而非经济属性

  • 世界上不存在绝对的产权，两个极端，完全不尊重权利和权利丝毫不能侵犯。
  • 权利不等于福利
  • 中国经济成就源于人权大幅改进

小结： 广告拦截软件有没有侵权？正在进行时，adblock胜诉

产权：使用权、收益权和转让权
  • 使用权和所有权，一份资产，两种说法，各自表述、互不影响。
  • 收入分配权（收益权）
  • 转让权：只有当他完全拥有一份资产的时候，他才能一次性的，把一项财产未来每一个时刻的使用权和收益权都转让出去。
产权保护之一：财产原则
  • 《财产原则，责任原则与不可让渡性：大教堂的一个视角（Property Rules， Liability Rules， and Inalienability：One View of the Cathedral》
    • 财产原则：一个人想要剥夺别人的所有权的话，他只有一个办法，就是向原来的所有权持有者付费，付到原来的所有权持有者愿意放弃为止。如果只有这种办法才能把所有权从别人手里拿过来的话，那么我们就说这种产权是根据“财产原则”进行保护的。
    • 责任原则：当一个人侵害了别人的产权以后，侵害者就要向原来的产权所有者进行赔偿，但是赔偿的金额不由原来的所有者定，而由第三方定。
    • 不可转让原则：政府禁止所有权人把他所拥有的资产卖给别人。
  • 哪种产权保护原则更好？
  • 由于交易费用，大量的产权不能通过“财产原则”保护
  • 为什么要惩罚刑事犯罪分子？

018-该不该征收拥堵费？

  • 答第一个问题：反对。如果价格这种方式可以合理配置资源，促进社会效率增加，那么我们就可以用，但是在拥堵费上，穷人一定比富人更加心疼钱，但我们能说富人比穷人更有需求出行吗？可能他只是有钱任性而已，这样资源其实未得到合理利用。
  • 知乎上回答补充了第一个问题，关于如果不适用价格竞争，那么我们采用哪种方式来竞争路权？我们如何解决拥堵这个问题？
  • 答第二个问题：不同意。时间和金钱对不同的人来说，成本是不一样的，穷人最不缺的就是时间，时间很便宜，但富人就不一样，所以两者的成本在本质上并不是一样的。

公用品和私用品
  • 公用品与私用品的关键区别：一个人用，别人就不能用的商品叫私用品；一个人用不影响别人使用的商品叫公用品。
  • 治安服务为什么是公用品？警察所提供的治安服务具有全局性、共享性。
  • “有竞争性”与“没有竞争性”（公用品和私用品）：一个人用，别人能不能用？
  • “排他性”和“非排他性”：一件商品它在实际使用的时候，物主能不能够很方便地把其他的使用者排除在外。
  • 一张表格看懂公用品和私用品 | | | | | :– | :– | :– | | | 排他性 | 非排他性 | | 竞争性 | 衣服、汽车、文具 | 鱼塘、地下水、森林 | | 非竞争性 | 有线电视、收费软件、著作版权 | 无线广播、开源软件、科学定律 |

  • Samuelson：灯塔市公用品，应由政府免费提供。政府有责任提供公用品。
  • Coase：《经济学当中的灯塔》 The Lighthouse in Economics，1974.
  • 商人乐于提供公共物品，比如大的购物广场内部空间
  • 灯塔之争的两大启发
    • 哪怕是公共设施，私人也有积极性提供，私人也能够从中收费。
    • 政府提供的公共服务，尤其是政府提供的私用品的服务，如果收费能够达到两个效果。
      • 他可以筛选需求，比如道路收费
      • 能够进行经济核算，比如修隧道，高铁

  • 思考：要不要不开放保护国内弱小的生产者？不去竞争永远弱！
贸易保护的理由辨析
  • 国家安全。这个理由成立，但还是要具体问题具体分析，比如在军事上，许多国家军队还往往用国外生产的武器，国外生产的通讯设备，到了平民的电脑产品，通讯设备，怎么就不行了？
  • 扶持幼年工业。真实的经验教训是那些要扶持的幼年工业，越扶持就越长不大，最后变成年迈的幼年工业。
  • 把钱留在本地（spend locally）。这是拒绝大范围合作分工，回退到小范围的合作分工。
  • 长期不练会生疏。你要学心算，将来没有计算器那怎么办？长期不种地，种地生疏怎么办？不需要妈的！
  • 工作流失。Trump“让美国再次伟大”把工作岗位找回来。美国工人应该在新的国际分工和合作的格局中，找到自己的位置。
  • 贫富分化。与现实不相符，自由贸易使先进国家的资本，跟落后国家的廉价劳动力结合起来，越是那些贫困的国家，越是那些有大量廉价劳动力的国家，他们在开放、分工、合作、自由交易的过程中，得到的好处就越大。那些发达国家的劳动力怎么办？只能不断地学习、不断改进、不断自我迭代、而不是故步自封，企图通过阻止竞争来保住自己的地位。
小结：为世界打工，让中国繁荣。不应该永远打工，要动态的而不是静态的考虑问题，随着时间的变化，比较优势在变化。贸易自由不需要对等。打工是发展的开始，生活只会越来越好。小米和华为就不是贸易保护出来的。

024-生产者（供给侧）的行为规律

成本与产出
  • 价格由需求量与供应量共同决定（供给曲线）
    • 价格是由一条倾斜向下的需求曲线和另外一条倾斜向上的供应曲线共同决定的。
    • 价格太高，供应不足，应通过技术的发展、资源的重新调配，供给曲线向右移动，也就是整个社会的生产能力得到提高。
    • 为什么铜的供应不断提高，但价格却几乎不变。
  • 事前看和事后看。再大的企业、再聪明的企业家、再多的数据，在做决策的时候，他们都没办法避免猜测的本质，否则那些有大量研究人员的企业，他们就永远不会失败了。比如诺基亚。
  • 决策就是拍脑袋
  • 长期与短期。短期是指只有个人的生产要素可以调整，大量其他的生产要素不能调整的情况；长期是指所有的生产要素都可以进行调整的情况。（不太明白）
  • 经营与歇业：什么时候继续经营，什么时候关门大吉。固定成本和可变成本。收入在可变成本与总成本之间，可以保持赤字经营；如果收入不足可变成本，就可以关门大吉了。
  • 生产速率与生产总量的区分。如果总产量不变，随着生产率的不断提高，平均成本最终就会增加。Haste makes waste。从长计议，慢慢规划，平均成本就会降下来。如果生产速率不变，而生产的总量越大，那么平均成本就越小。如果你每天很急着要打一个滴滴专车，那么长远来看，你就应该买个车。（不太明白）
投资加速器效应
  • 第一年的销量变化是10%，投资率则是100%，后面的总投资变化率没搞懂什么意思？
  • 为应对短暂的、瞬时的销售量变化，而投资的机器是耐用品。奥运会
  • 投资加速器的现实版：暂时不投资，观察下情况。等一等、看一看，增加一点库存，延长一点机器的折旧时间、增加一点空置的时间等方法。
  • 人们作出的投资不是跟需求的增减挂钩的，而是跟需求增减的变化率挂钩的。
供应的真相
  • 盈亏是由全盘计划来核算的：商人在开业以前，先做一个全盘的预测，是一个计划，一个猜想，觉得他的产品只要生产成这个质量、只要生产成这个规模，他就能卖出去，就能赚钱。
  • 规模经济是向前探索出来的
  • 产业集群是供应的趋势
企业家精神
  • 如何衡量企业家精神？企业家精神就是那种不折不挠的精神，在不确定的情况下做出决策，在人人都不看好的时候，满怀希望往前冲的精神。
  • 科学家必须跟风险投资家、企业家、制造商、营销专家紧密合作，才能把理论推向实际，从想法变成产品。
  • 制度创新带来产业兴旺。
小结：人在什么时候最爱大手大脚？经济发展速度的加速度在增加的时候。

025-价格歧视对消费者的好处

定价的规律（受价者与觅价者）
  • 完全竞争状态与受价者。例如农民，市场上农民数量非常多（卖家多），他们生产的产品质量差距非常小（同质化），他们自己的产量跟整个市场的产量相比微不足道，因此他们只能被动地接受市场的价格。例如股票市场，你只能接受市场的价格，你就是受价者。
  • 不完全竞争状态与觅价者。每个卖家的产品是差异化的，他们对价格和产量具有一定的操控权。
  • 不完全竞争是市场常态——觅价者
价格歧视能给觅价者脱罪（不太懂全部的逻辑）
  • 觅价者他们追求利润最大化的那个产量和价格，比他们本来能够生产的量要低，比他们能够提供产品的价要高！这一部分本来可以生产，却没有生产的一部分，叫做“无谓损失”
  • 垄断产量（边际收益=边际成本的那一点）和效率产量（边际成本＞边际收益）
  • 价格歧视可消除无谓损失。产商对产品进行区别定价，而不是统一定价。
价格歧视的策略
  • 完美的价格歧视，指的是卖家把交易剩余全占有了，全夺走了。例如一个饥饿的人，一口一口卖给他食物，第一口你愿意出的最高价100，第二口99，第三口98，占有全部的交易剩余。
  • 二级价格歧视，对不同的购买量收取不同的价格，多买就便宜一点，多买多送。但衡量到底买多少太麻烦。
  • 三级价格歧视，对不同的人群或不同的地域，收取不同的价格。如学生版、家庭版，专业版，终极版。
  • 越难以阻止消费者转售的产品，实施价格歧视的难度就越大。比如共享印度youtube账号。
行业竞争程度不看企业个数，看市场准入！
  • 企业合并是为了应对更激烈的市场竞争
  • 行业规模大小由企业家摸索决定
  • 有竞争必有成败，有成败必有企业退出市场
  • 竞争激烈程度的衡量标准：市场准入！竞争程度到底激不激烈，不看企业的个数，只看行业入口有没有人为设置的障碍
小结：飞机上的乘客谁补贴了谁？

都付出了成本，得到了自己想要的，这是歧视定价的作用，因为生产资源需要得到充分的利用。

觅价者比受价者更常见；“完全竞争市场”并不值得追求；价格歧视为觅价者脱罪（充分的生产，歧视的定价）；价格歧视的本质是交叉补贴，各取所需。

第二季度总结：人类如何增进彼此的幸福？

  • 选择、成本、需求、价格
  • 权利、时间、耐心、分工
  • 博弈产生权利，权利影响动机。 权利是大家愿意给你的，是人赋的，是从不断的冲突、博弈、犯错中，吸取教训而来的。自由与免费（free）一个政治权利，一个经济权利。使用权、收益权、转让权。
  • 耐心决定利率，利率影响价值。 耐用品和易耗品。
  • 消息影响价格，价格随机行走。 有效市场假说。
  • 风险影响收入，保险可买平稳。
  • 分工提高产量，交易增进幸福。 比较优势原理。

第三季度总结：信任、合作和竞争。

  • 建立信任就是减少意外。
  • 主动付出容易取信于人，恰当补偿可以激发忠诚。
  • 汉德公式：成本越低责任越大，避免意外所要付出的成本越低的一方，应担负起的责任就越大。
  • 管理的核心就是防止资源被欺负。
  • 强行调节工资会破坏企业活力。
  • 不公平的合约可以纠正不公平的关系。
  • 清晰的关系能使企业活力大增。
  • 有效才能做大，而不是做大了就会有效。
  • 技术进步史就是一部捆绑的历史。
  • 垂直限制能够促进品牌之间的竞争。
  • 倾销策略容易自毁长城。
  • 反垄断法能可错过，不可杀错。
