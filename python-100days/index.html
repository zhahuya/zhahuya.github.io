02day：变量、类型、运算符

运算符从高到低排列：

运算符 描述

03day：分支结构

  • if else
  • elif 这个需要理顺里面的逻辑关系，结构有点把握不清。
    • 解构这个函数能做到，但是正向使用有点迷。

04day：循环结构

  • for in 循环：如果明确知道循环执行的次数或者要对一个容器进行迭代。
  • while 循环：如果要构造不知道具体循环次数的循环结构。
  • break：终止它所在的这个循环。
  • continue：用来放弃本次循环后续的代码直接让循环进入下一轮。

05day：构造程序逻辑

  • 在学习了Python的核心语言元素（变量、类型、运算符、表达式、分支结构、循环结构等）之后，必须做的一件事情就是尝试用所学知识去解决现实中的问题。
  • 【换句话说】就是锻炼自己把用人类自然语言描述的算法（解决问题的方法和步骤）翻译成Python代码的能力，而这件事情必须通过大量的练习才能达成。
  • 这一张培养的是一种思维能力：将算法翻译成代码的能力。而将自然语言翻译成算法的能力，则是一种数学能力。
