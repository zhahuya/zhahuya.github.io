前言

最近这几天总感觉对自己时间和精力的掌控力降低了许多许多，因为工作的地方几个科室在一起，实在是很嘈杂，工作多倒还在其次；休息的地方，也是两个人空间，我是挺需要一个极度安静的空间的。

2020年的第一季度，总体分为3个阶段：上班（1.1~1.23） 过年（1.23~2.24） 上班（2.24~3.26）

1.23日是大年二十九，回家开始了自己的新年生活，不过到了初一，这新年就被一件足以载入史册的疫情打破了，人们被迫宅在家里，不过这正好顺了我的意。直到2.17日，终于我城的封锁稍微放开了一些，不过还是很麻烦，需要开很多证明，而且出去了就不准你再回来，不过还是要上班吧，来到这里，隔离了7天，也就是2.24号，从那天到今日，正好是一个月。所以这第一季度就是上班-过年-上班的循环，包括年前的一次调岗，这些不稳定对我自己稳定的时间掌控冲击还是挺大的。

下边讲一讲这一季度都干了点什么吧。

计算机科学

  1. 重新把 < Crash Course Computer Science> 看了一遍，并做了笔记，这次比第一次收获要大一些，其中描述“变量”其实是内存中的一个位置，对我简直是醍醐灌顶的发醒。
  2. 《小甲鱼的Python课程》，这是让我感到挫败的第一件工作，从1.13号开始的，按照计划，它应该结束的，但是到今天大概完成了进度的70%，不过也不能苛责自己太多，已经很棒了，接下来的一个月内，要将它完成，即4月底完成这项任务。
  3. <Code: the hidden language of computer hardware and software>，这本书啃的生肉，有点难，进度也不喜人，大概完成了30%，但书还是非常值得一读的，可以安排到下个季度。
  4. 众多计算机知识的应用：博客的搭建，网站的搭建，服务器的选择与购买，域名的选择与购买。
  5. 前端三大框架 HTML + CSS + JavaScript 的整体掌握，electron语言的学习，需要在下一季度进行安排。下一季度 预计完成 Python + 前端三大的第一次掌握。

阅读与写作

  1. 阅读《数学通识》，收获巨大，真的让我重新认识了数学，一门我从来没看清过的学科，真正的理解了什么叫做思维的体操，虽然我很菜，但我很想说：“我爱数学”。
  2. 看了一下《线性代数》《微积分》，虽然大学学过高数，但到了今天，才终于知道这两个到底是什么东西，不得不说这种教育真的害死人。
  3. 阅读《武志红心理学课》，这是让我感到比较挫败的一件工作，从1.2号开始的，到今天3月底了，还有大概1/4未读完，按照我的计划，是不应该这样的。这也和我批评其中的一些章节，比较模糊个人有关，用作者自己的话说就是“玄乎”，不过我认为这是作者能力不足，无法驾驭一些概念，及一些理论无法完全解释现实世界，或者是他未找到一种纲举目张的“纲”，可能与作者思想的高度有关。虽然说了这么多“坏话”，我对这门课程的总体评价还是很高的，从科普的角度来说，是顶尖的了吧，而且对我很有帮助，很有益处。
  4. 写了许多读书笔记，原创的文章较少，主要是没什么好的议题，有很多的议题也是不能说。在写作的逻辑思维方面，需要进一步的加强，这部分甚至可以依赖输入好的信息来获得，但是写作的方法和技巧则是需要自己亲自去写，去表达自己的想法才能锻炼的。

公考

  1. 完成了《资料分析》的课程学习，练习进度完成了预期的55%，这不那么令我感到挫败，因为这玩意太恶心了，谁愿意搞这个呢，已经很棒了！还可以做得更好，但是需要更好的环境，需要我去帮自己创造。
  2. 《数量关系》课程的学习，按照计划，只完成了30%便夭折了，数学还是不那么令人感到有趣，尤其是被拿来考别人的时候，实在是很糟糕！
  3. 下一季度，《判断推理》可以安排上了，练习需要安排上了，需要更加可靠的环境，一定要去争取一个适合学习工作的环境。

英语

  • 没有什么特别的计划，顺其自然，在使用中逐步提高吧。

经济

  • 有舍才有得，不要舍不得对自己的投资，顺从自己的想法，大胆地去做。

娱乐

bilibili and youtube

  • 看了老蒋、妈咪说等一批up主精彩的视频，令我获益匪浅。

王者荣耀

  • 王者荣耀大概花了我10天的业余时间，终于单排到了王者，上王者对我来说还是挺有意义的，对我来说就是一个实力的证明，以前虽然胜率高，但打的少没王者，总让人觉得没那个实力。

  • 我主玩打野、其他ADC，中单，辅助都会玩，但是上单玩的少一点。

  • 王者上分的要点：恃强凌弱（经济）；以多打少（人数）；稳住心态。

    • 星耀以下恃强凌弱：主要是抢经济，迅速发育，一般我经济都全场最高，老年人手法也能一打2，这一点都不精妙，就是经济压制，我多出的经济大部分是从对方的野区和河蟹得到的，对对方的经济的压制也很重要。
    • 星耀以上以多打少：主要是联合中单和辅助，以多打少，一般这两个至少有一个聪明的，一个是呆瓜，这样一般都能赢，两个都聪明就是碾压，两个都是呆子就输。
    • 不管那个段位都要稳住心态，其一不要急躁，要相信，只要自己处理好，任何一场团战都是有机会的；其二要爱队友，要知道再呆的队友，发出的技能也是伤害敌人的，就算打不到敌人也能封路，多给队友点爱吧，谁都有失误，输多了就冷静一天再打！
