37-类和对象：介绍对象

  • 备注：2020.03.21日重启学习

  • 对象是什么？对象的特征

  • 封装：把一堆数据扔进列表里封装起来；把一些常用的代码块打包成函数；

  • 对象：模拟真实世界，把数据和代码封装在一起。

  • 小乌龟：外形 + 动作

    • 外形：绿色；四条腿；10KG；有外壳；大嘴巴
    • 动作：爬、跑；咬人；吃东西；睡觉

  • 对象 = 属性（变量） + 方法（函数）

  • 类（Class）：使对象达到量产的效果，对象是类的一个实例（instance）

    • 类是图纸，根据一张图纸可以设计成千上万的房子，即实例对象
    • 类约定使用大写字母开头

  • OO（面向对象）的3个特点：Object Oriented

    1. 封装。封装是一种信息隐蔽技术。例如，对于一个【列表】来说，我们可以使用列表的各种方法，其实【列表】就是一个对象，我们使用的方法，我们不知道是如何具体实现的，那其实是封装的函数。
    2. 继承。继承是子类自动共享父类之间数据和方法的机制
    3. 多态。多态是不同对象 对 同一方法响应 不同的行动。

    • 其实就是把变量改了名字：_Class__variable（变量名）
    • 其实是伪私有，没有设置权限，只是改了名字

39-类和对象：继承

  • 鱼类：金鱼（goldfish）、鲤鱼（carp）、三文鱼（salmon）、鲨鱼（shark）

  • 继承：不需要每次都定义一个新的鱼类，大部分鱼的属性和方法都是相似的。

  • 父类、基类、超类 —— 子类

  • class DirivedClassName( BaseClassName )，例如：class Child(Parent)

  • 如果子类中定义与父类同名的方法或属性，则会自动覆盖父类对应的方法或属性

  • 调用未绑定的父类方法

  • 使用super函数

  • 多重继承：class DirivedClassName(base1, base2, base3)，应尽量避免使用它。

40-类和对象：拾遗、组合

  • 组合：横向的，没什么关系的类在一起

  • 类、类对象和实例对象

    • 不要试图在一个类里边 定义出所有能想到的特性和方法，应该利用继承和组合机制来进行扩展。
    • 用不同的词性命名，如属性名用名词，方法名用动词。

  • 什么是绑定？

    • Python严格要求方法需要有实例才能被调用，这种限制就是Python所谓的绑定。

41-类和对象：一些相关的BIF（builtin)

  • issubclass(class, classinfo)：是否是子类
  • isinstance(object, classinfo)：是否是类的实例
  • hasattr(object, name)：attribute 属性、特征
  • getattr(object, name[, default]
  • setattr(object,name,value]
  • delattr(object,name)
  • property(fget = None, fset = None, fdel = None, doc = None)
