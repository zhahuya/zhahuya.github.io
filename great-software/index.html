我在使用的软件分享

2020-04-04
应用技术
约 1321 字 预计阅读 3 分钟

文章目录

  • 前言
  • windows
  • 移动端：android、ipad

前言

  • 之前发在xx的，没有使用 Markdown 排版实在是太糟糕了，现在把它重新写一下。
  • 这些软件的共同特点就是极其小，几乎都是5M以下，中位数可能在2M，但是功能极其强大。
  • 这些软件实在是太棒了，简洁、美观、强大，可以极大的提高你的工作体验和效率！

windows

  1. 截图软件：snipaste

    • 非常强大的截图软件，有很多工具，十分适合在截图上进行编辑。
    • 将快捷键设置为alt+a，最大化还原操作习惯。

  2. 快速搜索工具：listary

    • 双击Ctrl快速呼出，什么都可以搜，速度极快，软件本身极其小巧，3.5M。
    • 我还试用了火萤酱这款软件，太大了100多M，功能臃肿，反应不灵敏，一种让人感觉很强大，却不能提升效率，反而增加了干扰的二流软件。

  3. 文件查找工具：Everything

    • 精致小巧绿色，搜东西太快了，好用。
    • 已经使用了近半年，确实非常好用，但是现在要和 Listary 比较一下，看它们的功能是互补还是冲突，考虑删除其中一个。

  4. Windows预览工具：Quick Look

    • 空格快速预览几乎所有文件，很方便，可能适合 Mac 转过来 windows 的用户。
    • 我没有预览文件的习惯和需要，从近半年的实际使用来看，用到的极少。

  5. 英语软件：欧陆词典；词根词缀词典

    • 将快捷键设置为alt+w，可以在遇到全屏时调出主程序来查词，效率很高。
    • tips：windows10的win+上下左右分屏效果，非常棒！
    • 词根词缀词典简直是梦中的软件，有详细的词根词源的解释，如果能在PC上使用就好了，现在只有网页版。

  6. 卸载工具：geek uninstaller

    • 一款专注于卸载的软件，非常的小巧，在功能上卸载真正做到了干干净净，明明白白，有人会问一个管家、卫士不就搞定了吗？它们都太臃肿了！

  7. 视频录制软件：Apowersoft

    • 对比了我以前喜欢用的 bandicam ，和它同样好用，但比它还要强大，可以进行视频中的编辑编著。
    • 在PJ方面，比 bandicam 要好很多，bandicam 的致命缺陷开个梯子就失效了。

  8. 大纲与思维导图工具：幕布

    • 可以一键生成思维导图的大纲笔记工具，收费但是有很多的免费体验时间。
    • 一款有序、精美的写字软件，已经用了4个月，大大改善了我的写作、笔记体验。

  9. 思维导图：Xmind Zen

    • 很好用的一款软件，和幕布是一个互补的关系。

  10. 视频播放器：PotPlayer

    • 这绝对是最强大的播放器，无广告、免费、小巧。

  11. PDF阅读器：SumatraPDF

    • 解析速度比常见的几个商业软件都要快一点，喜欢！

  12. 流程图: Diagram Designer

    • 一款娇小(2M)的开源软件,简单强大.
    • 原生的中文. ​

移动端：android、ipad

  1. 时间管理软件：番茄todo

    • 收费，但是超级便宜49.9三年。
    • 已经用了一年多了，现在是越来越爱，而且它也变得越来越好了。

  2. 微博第三方客户端：share

    • 无广告且功能够用，让你轻松无干扰的获取信息，使用了近半年，已经完全放弃官方客户端了。

  3. 学英语：轻听英语

    • 终身会员168，双十一会打折。
    • 软件设计十分适合精听，有了这个软件，钟道隆老先生不用学坏几个收音机了。
    • 之前界面很清新友好，最近升级变得杂乱，一进去就得看发现和推荐版块，体验减一分！
    • 有书虫系列，加一分。但是近期好像被整改了，很多资源都下架了，减超多分。
    • 目前来看，学英语还是推荐 youtube ，上面有很多免费的制作精良的课程。

  4. Rss订阅软件：feedly

    • 自己订阅点信息来浏览，适合关注个人博客的人使用，一般人应该没这个需求了。
